# 06-paso4_instrucciones.md

## Paso activo
Direccion creativa para imagenes

## Objetivo del paso
Traducir el mensaje de campana en prompts visuales de alta calidad para producir creatividades.

## Salida obligatoria
Responder solo con esta tabla:

| ID creativo | Objetivo pieza | Concepto visual | Prompt de imagen final | Formato | Uso sugerido |
|---|---|---|---|---|---|

## Reglas
- entregar minimo 3 prompts visuales
- cada prompt debe incluir estilo, composicion, luz, texto en pieza y tono emocional
- no usar lenguaje ambiguo en `Prompt de imagen final`
