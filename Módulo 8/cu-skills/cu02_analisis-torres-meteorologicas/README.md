# CU02 — Análisis de datos de torres meteorológicas SAE

Este caso de uso contiene los materiales para construir y probar un skill operativo que revisa datos CSV de sensores de torres meteorológicas, detecta huecos de datos, valores fuera de rango (outliers) y sensores congelados.

## Ruta recomendada

1. Lee `00_ficha.md` para entender el caso.
2. Sigue `02_paso-a-paso.md` para construir el skill.
3. Usa los archivos de `insumos/` cuando el paso a paso lo indique.
4. Copia los mensajes de `prompts/prompts-de-uso.md` para probar el skill.

## Contenido de la carpeta

| Ruta | Contenido |
|------|-----------|
| `00_ficha.md` | Contexto, objetivo, valor y criterios de éxito |
| `01_ejercicio.md` | Guía del ejercicio para el participante |
| `02_paso-a-paso.md` | Instrucciones para construir y probar el skill |
| `insumos/` | CSV ficticio de sensores de torre meteorológica SAE |
| `prompts/` | Prompts listos para invocar el skill |
| `configuracion/` | Descripción del skill para el @skill-creator (Iteración 1), corrección de formato (Iteración 2) y fallback manual |

## Orden de trabajo

Empieza por la ficha, continúa con el ejercicio y utiliza el paso a paso como guía operativa.
