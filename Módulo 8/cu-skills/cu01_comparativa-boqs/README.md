# CU01 — Comparativa inteligente de BOQs SAE

Este caso de uso contiene los materiales para construir y probar un skill operativo que analiza ofertas de licitación (BOQs) y detecta automáticamente desviaciones, partidas omitidas y precios anómalos.

## Ruta recomendada

1. Lee `00_ficha.md` para entender el caso y el valor del skill.
2. Lee `01_ejercicio.md` para conocer los pasos y el resultado esperado.
3. Sigue `02_paso-a-paso.md` para construir el skill paso a paso con los prompts exactos.
4. Usa los archivos de `insumos/` cuando el paso a paso lo indique.
5. Consulta `prompts/prompts-de-uso.md` para invocar el skill y probar variantes.

## Contenido de la carpeta

| Ruta | Contenido |
|------|-----------|
| `00_ficha.md` | Contexto, escenario, alcance del skill y valor esperado |
| `01_ejercicio.md` | Guía del ejercicio para el participante + resultado de referencia |
| `02_paso-a-paso.md` | Flujo de 8 pasos con prompts listos + banco de respuestas para el formador |
| `insumos/` | BOQ ficticio con anomalías sembradas (Planta P-07, 3 licitadores) |
| `prompts/` | Prompts listos para ejecutar el skill y sus variantes |
| `configuracion/` | Instrucciones del skill como fallback si @skill-creator no está disponible |

## Orden de trabajo

Empieza por la ficha, continúa con el ejercicio y utiliza el paso a paso como guía operativa.
