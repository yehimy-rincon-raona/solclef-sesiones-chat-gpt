# 02 - Crear perfil del viaje

Con base en mis respuestas, crea el perfil del viaje.

Antes de escribir el perfil, revisa si tienes informacion suficiente para completar estos campos:

- tipo de experiencia deseada;
- duracion estimada;
- presupuesto;
- nivel de comodidad esperado;
- restricciones importantes;
- prioridades del viaje;
- destinos que podrian encajar.

Si falta algun dato importante, no inventes.

Primero hazme preguntas de aclaracion con opciones de respuesta.

Usa este formato:

## Datos pendientes para crear el perfil

1. Que dato necesitas aclarar?
   A. Opcion posible
   B. Opcion posible
   C. Opcion posible
   D. No estoy seguro/a
   E. Otra / especificar

Haz maximo 5 preguntas de aclaracion.

Cuando tenga informacion suficiente, crea el perfil usando el formato de `template-perfil_del_viaje.md`.

Si aun queda algun dato incompleto, marca el campo como "Pendiente" y explica brevemente que falta.

Todavia no elijas destino final.
