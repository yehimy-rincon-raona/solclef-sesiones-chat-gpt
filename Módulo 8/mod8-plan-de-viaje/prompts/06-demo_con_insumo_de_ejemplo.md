# 06 - Demo con insumo de ejemplo

Este prompt sirve para ejecutar una demostracion rapida usando el archivo `insumos/ejemplo-respuestas_viajero.md`.

```text
Usa el archivo `ejemplo-respuestas_viajero.md` como si fueran mis respuestas iniciales.

Ejecuta el caso completo en version breve:

1. Crea el perfil del viaje.
2. Propone entre 3 y 5 destinos.
3. Compara los destinos en tabla.
4. Crea una matriz de decision simple.
5. Recomienda un destino ganador.
6. Indica que datos se deben validar antes de reservar.
7. Propone el siguiente paso para empezar itinerario y presupuesto.

Mantén la respuesta clara, breve y escaneable.
```
