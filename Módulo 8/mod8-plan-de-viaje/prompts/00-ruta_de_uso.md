# 00 - Ruta de uso de prompts

Este archivo indica en que orden usar los prompts del caso.

La idea es copiar cada prompt en el Proyecto de ChatGPT y avanzar paso a paso.

## Orden recomendado

### Paso 1

Usa:

```text
01-iniciar_descubrimiento.md
```

Objetivo: activar la fase de preguntas para entender el viaje.

### Paso 2

Usa:

```text
02-crear_perfil_del_viaje.md
```

Objetivo: convertir las respuestas del usuario en un perfil resumido.

### Paso 3

Usa:

```text
03-comparar_destinos.md
```

Objetivo: comparar entre 3 y 5 destinos posibles.

### Paso 4

Usa:

```text
04-matriz_de_decision.md
```

Objetivo: evaluar los destinos con criterios ponderados.

### Paso 5

Usa:

```text
05-elegir_destino_y_siguiente_paso.md
```

Objetivo: elegir el destino recomendado y dejar listo el inicio de planeacion.

## Variante rapida

Si quieres mostrar el ejemplo en pocos minutos, usa:

```text
06-demo_con_insumo_de_ejemplo.md
```

Ese prompt usa el archivo `insumos/ejemplo-respuestas_viajero.md` para simular un caso completo.
