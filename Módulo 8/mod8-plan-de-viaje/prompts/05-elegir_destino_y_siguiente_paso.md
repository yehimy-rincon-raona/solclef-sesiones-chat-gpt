# 05 - Elegir destino y siguiente paso

Copia y pega este prompt despues de la matriz de decision.

```text
Con base en el perfil del viaje, la comparacion de destinos y la matriz de decision, ayudame a cerrar la decision.

Entrega:

1. Destino recomendado.
2. Por que es la mejor opcion.
3. Que tipo de experiencia se puede construir.
4. Mejor temporada o fechas sugeridas.
5. Datos que debo validar antes de reservar.
6. Siguiente paso recomendado para planear itinerario, presupuesto y reservas.

No armes todavia el itinerario completo. Deja solo una ruta clara para la siguiente conversacion.
```
