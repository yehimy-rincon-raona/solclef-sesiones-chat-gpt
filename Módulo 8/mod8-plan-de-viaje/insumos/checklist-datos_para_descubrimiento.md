# Checklist - Datos para descubrimiento del viaje

Usa este checklist para saber si el Proyecto ya tiene suficiente informacion antes de recomendar destinos.

- [ ] Ciudad o pais de salida.
- [ ] Numero de viajeros.
- [ ] Duracion aproximada.
- [ ] Fechas exactas o temporada.
- [ ] Presupuesto estimado por persona.
- [ ] Tipo de experiencia deseada.
- [ ] Nivel de comodidad esperado.
- [ ] Preferencia nacional o internacional.
- [ ] Destinos en mente.
- [ ] Destinos descartados.
- [ ] Restricciones de visa o pasaporte.
- [ ] Restricciones de idioma.
- [ ] Restricciones de salud o movilidad.
- [ ] Consideraciones de seguridad.
- [ ] Ritmo del viaje.
- [ ] Estilo de planeacion.

## Criterio de avance

Si faltan mas de 4 datos clave, el Proyecto debe seguir preguntando.

Si falta informacion menor, puede avanzar con supuestos marcados como pendientes.
