# 00 - Estructura recomendada del Proyecto en ChatGPT

## Nombre sugerido

Mi viaje ideal

Otros nombres posibles:

- Destino perfecto
- Planeador de viajes inteligentes
- Explorador de destinos
- Viaje desde cero

## Objetivo del Proyecto

Ayudar a una persona a elegir su destino ideal antes de crear itinerarios, presupuestos o reservas.

El Proyecto debe guiar al usuario desde una idea inicial de viaje hasta una recomendacion clara de destino, usando preguntas, comparacion de opciones y una matriz simple de decision.

## Archivos sugeridos para cargar

- `config/01-instrucciones_del_proyecto.md`
- `templates/template-perfil_del_viaje.md`
- `templates/template-comparacion_destinos.md`
- `templates/template-matriz_decision.md`
- `insumos/ejemplo-respuestas_viajero.md`

## Flujo general

1. Descubrir el contexto del viaje.
2. Crear el perfil del viaje.
3. Comparar entre 3 y 5 destinos.
4. Evaluar con matriz de decision.
5. Elegir destino recomendado.
6. Definir siguiente paso de planeacion.

## Uso didactico

Este caso esta pensado como ejemplo basico para mostrar:

- como configurar instrucciones en un Proyecto de ChatGPT;
- como cargar plantillas de salida;
- como usar prompts secuenciales;
- como mantener contexto entre conversaciones del mismo Proyecto;
- como separar descubrimiento, analisis y decision.
