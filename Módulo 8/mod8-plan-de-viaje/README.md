# Modulo 8 - Proyecto basico: Planeador de viajes

Este caso convierte la idea del archivo `proyecto-planeador-de-viajes-inteligente.md` en una estructura lista para crear y usar un Proyecto en ChatGPT.

El objetivo es practicar el uso basico de Proyectos: instrucciones, archivos de apoyo, prompts por fases y plantillas de salida.

## Como usar esta carpeta

1. Crea un Proyecto nuevo en ChatGPT.
2. Usa como nombre sugerido: `Mi viaje ideal`.
3. Copia el contenido de `config/01-instrucciones_del_proyecto.md` en las instrucciones del Proyecto.
4. Sube o consulta los archivos de `templates/` e `insumos/` como material de apoyo.
5. Abre `prompts/00-ruta_de_uso.md` y sigue los prompts en orden.

## Estructura

- `config/`: instrucciones generales para configurar el Proyecto.
- `prompts/`: prompts paso a paso para ejecutar el caso en ChatGPT.
- `templates/`: formatos de salida esperados.
- `insumos/`: ejemplos simples para practicar.
- `proyecto-planeador-de-viajes-inteligente.md`: documento base original.

## Archivo clave para preguntas

Para que el descubrimiento sea mas facil en clase, usa:

```text
templates/template-preguntas_con_opciones.md
```

Ese archivo muestra como hacer preguntas con opciones de respuesta.

Para aclarar datos antes de crear el perfil, usa:

```text
templates/template-aclaraciones_para_perfil.md
```

## Resultado esperado

Al terminar el ejercicio, el usuario debe tener:

1. Perfil del viaje.
2. Lista corta de destinos recomendados.
3. Comparacion objetiva entre destinos.
4. Matriz de decision.
5. Destino recomendado.
6. Siguiente paso para armar itinerario, presupuesto y reservas.
