# Prompt 03: Investigar cómo crear el Skill

## Objetivo

Usa este prompt después de elegir un tema de Skill.

Sirve para investigar qué necesita ese Skill, cómo debería funcionar y qué referencias conviene reunir antes de diseñarlo.

---

## Prompt principal

```text
Ya elegí este tema de Skill:

[describe el tema del Skill]

Ayúdame a investigar cómo debería crearse este Skill antes de diseñarlo formalmente.

Usa las guías de investigación, diseño y evaluación del proyecto para analizarlo.

Después, entrégame:

1. Síntesis de investigación.
2. Supuestos detectados.
3. Preguntas pendientes.
4. Recomendaciones para diseñar el Skill.
5. Decisión sobre si conviene avanzar al diseño.

No documentes el Skill todavía.
```

---

## Prompt para profundizar en recursos

```text
Con base en el tema del Skill, ayúdame a definir qué recursos debería cargar o preparar para que funcione mejor.

Usa los criterios del proyecto para clasificar recursos obligatorios, opcionales y futuros.

Incluye referencias, plantillas, ejemplos, criterios, checklists o automatizaciones solo si realmente aportan valor.
```

---

## Resultado esperado

Al terminar este paso deberías tener:

- Una investigación preliminar del Skill.
- Preguntas pendientes.
- Recursos necesarios.
- Criterios para pasar al diseño.
