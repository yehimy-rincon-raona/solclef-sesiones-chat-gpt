# Prompt 02: Descubrir el tema del Skill

## Objetivo

Usa este prompt cuando todavía no sabes qué Skill crear.

Sirve para descubrir oportunidades a partir de tus tareas, procesos, dolores, repetición de trabajo o necesidades reales.

---

## Prompt principal

```text
Quiero descubrir qué Skill podría crear.

Usa la metodología del proyecto para guiarme paso a paso.

Primero hazme preguntas para identificar tareas, procesos, necesidades, entregables o dolores que podrían convertirse en Skill.

Después de mis respuestas:

1. Agrupa las oportunidades detectadas.
2. Propón posibles temas de Skill.
3. Evalúa cada tema usando los criterios del proyecto.
4. Recomienda los 3 mejores candidatos.
5. Indica cuál conviene desarrollar primero y por qué.

No documentes todavía el Skill. Primero ayúdame a descubrir el mejor tema.
```

---

## Prompt alternativo si ya tienes varias ideas

```text
Tengo estas ideas de posibles Skills:

[pega aquí tus ideas]

Evalúalas y ayúdame a elegir cuál conviene desarrollar primero.

Usa la matriz y criterios de evaluación del proyecto.

Termina con una recomendación priorizada.
```

---

## Resultado esperado

Al terminar este paso deberías tener:

- Un tema de Skill seleccionado.
- Una justificación de por qué vale la pena.
- Una primera idea del usuario, problema y resultado esperado.
