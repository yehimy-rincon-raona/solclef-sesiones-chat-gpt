# Prompt 05: Evaluar el Skill diseñado

## Objetivo

Usa este prompt cuando ya tengas una idea o diseño preliminar del Skill y quieras saber si está bien planteado.

---

## Prompt principal

```text
Quiero evaluar este diseño de Skill:

[pega aquí el diseño, resumen o documentación preliminar]

Evalúalo usando la guía de evaluación del proyecto.

Usa una matriz de evaluación con puntaje y justificación.

Termina con:

1. Diagnóstico general.
2. Fortalezas.
3. Debilidades.
4. Riesgos.
5. Recomendaciones de mejora.
6. Decisión: avanzar, rediseñar, simplificar o no convertir en Skill.
7. Próximos pasos.
```

---

## Prompt para mejorar el diseño después de evaluar

```text
Con base en la evaluación anterior, ayúdame a mejorar el diseño del Skill.

Usa las recomendaciones de la evaluación para ajustar los elementos débiles del diseño.

Indica qué información falta antes de documentarlo formalmente.
```

---

## Resultado esperado

Al terminar este paso deberías saber:

- Si el Skill es viable.
- Qué tan maduro está.
- Qué mejorar antes de documentar.
- Si conviene avanzar al template.
