# Prompt 04: Diseñar el Skill por fases

## Objetivo

Usa este prompt para diseñar el Skill con base en la metodología del proyecto.

Este paso convierte la idea en una estructura clara y evaluable.

---

## Prompt principal

```text
Quiero diseñar este Skill por fases:

[describe el tema, problema, usuario y contexto si ya los tienes]

Guíame usando la metodología de diseño del proyecto.

Para cada fase definida en la guía de diseño:

1. Explícame qué se está definiendo.
2. Propón una versión inicial.
3. Marca supuestos.
4. Señala preguntas pendientes.
5. Indica si el diseño está suficientemente claro para continuar.

No saltes directamente a la documentación formal.
Primero ayúdame a diseñar el Skill con solidez.
```

---

## Prompt fase por fase

Usa este prompt si prefieres trabajar una fase a la vez:

```text
Quiero trabajar solo la fase [nombre de la fase] del diseño de mi Skill.

Mi Skill trata sobre:

[describe el Skill]

Ayúdame a definir esta fase usando la guía de diseño del proyecto.

Entrega una propuesta inicial, supuestos, pendientes y criterios para saber si esta fase está lista.

No avances a la siguiente fase hasta que esta quede clara.
```

---

## Prompt para crear instrucciones tipo SKILL.md

```text
Con base en el diseño que ya construimos, redacta una primera versión de instrucciones para un posible `SKILL.md`.

Usa los criterios de diseño del proyecto para incluir activación, uso, límites, flujo, formato de salida y criterios de calidad.

Aclara qué partes son propuestas y qué partes están pendientes por validar.
```

---

## Resultado esperado

Al terminar este paso deberías tener:

- Diseño completo del Skill.
- Objetivo operativo.
- Casos de uso.
- Entradas y salidas.
- Flujo.
- Reglas.
- Recursos.
- Criterios de calidad.
- Riesgos y pendientes.
