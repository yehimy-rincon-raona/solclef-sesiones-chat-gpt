# Insumos para builder

Esta carpeta reúne los insumos listos para usar al momento de montar el GPT en el builder de ChatGPT.

## Contenido

- `01-builder_instructions_cortas.txt`
- `02-knowledge_north_star_builder_base.md`
- `03-ejemplo_instrucciones_clase.md`

## Para que sirve cada archivo

### `01-builder_instructions_cortas.txt`

Archivo corto pensado para copiar y pegar directamente en el campo `Instructions` del builder.

### `02-knowledge_north_star_builder_base.md`

Archivo largo pensado para subirse a `Knowledge` dentro del builder.
Contiene la base completa que el GPT debe seguir.

### `03-ejemplo_instrucciones_clase.md`

Documento de respaldo en formato legible para clase.
Sirve como ejemplo cuando una persona no alcance a construir el GPT en vivo o cuando quieras mostrar una version consolidada sin depender del ejercicio del grupo.

## Regla de uso

Primero se puede trabajar el caso en el chat normal durante la clase.
Despues, cuando ya exista suficiente claridad, se usa esta carpeta para montar el GPT asi:

1. subir `02-knowledge_north_star_builder_base.md` a `Knowledge`
2. pegar `01-builder_instructions_cortas.txt` en `Instructions`
3. usar `03-ejemplo_instrucciones_clase.md` como respaldo docente
