# Cierre y montaje

## Objetivo del paso

Cerrar la practica y dejar el GPT listo para montarlo en ChatGPT.

## Lo que debes hacer

1. Revisa el material guardado de las tres fases.
2. Confirma que el caso sigue siendo el mismo en todo el recorrido.
3. Verifica que la logica `North Star -> System Prompt` se mantenga.
4. Lleva la configuracion final al builder de GPTs en ChatGPT.

## Prompt de validacion final

Antes de montarlo, puedes pedir esta revision:

```text
Antes de cerrar, audita esta configuracion del GPT.

Quiero que me indiques:

1. que esta claro
2. que esta ambiguo
3. que partes del System Prompt no dependen suficientemente del North Star
4. que mejorarias sin cambiar el enfoque

Responde de forma breve, concreta y accionable.
```

## Checklist de cierre para clase

- ya se entiende el caso de uso
- ya existe una logica por fases
- el `North Star` esta bien definido
- el `System Prompt` sale de ese `North Star`
- el GPT no depende de invenciones
- el material sirve como insumo para explicar la metodologia

## Frase de cierre sugerida para la sesion

No construimos primero un prompt.
Construimos primero una definicion.
Y cuando la definicion estuvo clara, el prompt final salio con mucha mas coherencia.
