# Paso a paso de clase

Esta carpeta esta pensada como insumo directo para la sesion.
La idea es simple:

- yo actuo como maestra especialista en uso de ChatGPT
- tu actuas como alumna que sigue la secuencia paso a paso
- no improvisamos el flujo
- trabajamos una fase a la vez

## Objetivo de esta guia

Construir el GPT del Tema 1 dentro de ChatGPT siguiendo una ruta clara, repetible y facil de explicar.

## Como usar esta carpeta en clase

Sigue los archivos en este orden:

1. `00-guia_docente.md`
2. `00b-guion_clase_en_vivo.md`
3. `01-preparacion.md`
4. `02-fase_1_descubrimiento.md`
5. `03-fase_2_estructura.md`
6. `04-fase_3_configuracion.md`
7. `05-cierre_y_montaje.md`

## Regla de trabajo para la alumna

En cada paso debes hacer solo estas cuatro cosas:

1. leer la instruccion del paso
2. copiar el prompt sugerido
3. pegarlo en ChatGPT
4. guardar la salida util antes de pasar al siguiente paso

## Resultado esperado de la practica

Al terminar deberias tener:

- un caso de uso bien entendido
- un `North Star` definido
- una estructura conversacional clara
- un borrador de `System Prompt`
- un GPT listo para montarse o terminar de afinar en ChatGPT
- un guion de facilitacion con tiempos para conducir la practica en vivo
