# Ideas de GPTs internos Solclef - Material de practica

> Documento formativo ficticio para ejercicios de creacion de GPTs personalizados.  
> Las ideas se usan como ejemplos de aprendizaje y no representan herramientas oficiales de Solclef.

---

## Como usar este archivo

1. Selecciona una idea de la lista.
2. Lee el objetivo, usuarios, entradas y salida esperada.
3. Completa el prompt maestro de `configuracion/prompt-maestro.md`.
4. Usa el ayudante creador de GPTs para generar la primera configuracion.
5. Revisa y ajusta antes de guardar el GPT.

## Idea 1 - GPT Minutas Operativas

| Elemento | Descripcion |
|----------|-------------|
| Necesidad | Convertir notas sueltas de reuniones en minutas claras. |
| Usuarios | Responsables de area, equipos operativos, equipos de seguimiento. |
| Entrada | Notas, apuntes o transcripcion breve de una reunion. |
| Salida | Minuta con resumen, acuerdos, responsables, fechas y pendientes. |
| Limites | No inventar acuerdos, responsables ni fechas no mencionadas. |

Prompt corto de practica:

```text
Quiero crear un GPT que convierta notas sueltas de reuniones operativas en minutas estructuradas.
```

## Criterios para elegir una idea

| Criterio | Pregunta |
|----------|----------|
| Claridad | Se entiende que problema resuelve? |
| Alcance | Puede probarse en una conversacion corta? |
| Entrada | El usuario puede entregar la informacion necesaria? |
| Salida | El resultado se puede validar facilmente? |
| Riesgo | El GPT evita decisiones criticas o sensibles? |
