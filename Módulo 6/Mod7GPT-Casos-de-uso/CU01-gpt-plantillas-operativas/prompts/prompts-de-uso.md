# Prompts de uso - GPT Plantillas Operativas Solclef

> **Donde se usa:** como mensaje en una conversacion con el GPT Plantillas Operativas Solclef.  
> Documento formativo ficticio.

---

## Opcion A - Crear una plantilla desde una necesidad simple

```text
Necesito una plantilla simple para registrar una reunion mensual de seguimiento de planta. Debe incluir acuerdos, responsables, fechas y estado de avance.
```

---

## Opcion B - Crear una matriz de seguimiento

```text
Crea una matriz para hacer seguimiento de acciones pendientes de un equipo operativo. Debe incluir codigo de accion, responsable, fecha objetivo, estado y comentarios.
```

---

## Opcion C - Crear un checklist breve

```text
Genera un checklist para registrar una visita de seguimiento operativo a una planta. Debe ser breve, facil de completar en campo y permitir registrar observaciones.
```

---

## Opcion D - Crear una ficha de iniciativa

```text
Crea una ficha breve para documentar una iniciativa ESG. Debe incluir objetivo, alcance, responsable, estado, indicador de seguimiento y proximo paso.
```

---

## Opcion E - Ajustar una plantilla generada

```text
Haz la plantilla anterior mas simple. Reduce los campos al minimo necesario y mantenla en una sola tabla.
```

---

## Opcion F - Pedir una version mas detallada

```text
Amplia la plantilla anterior para que pueda usarse en una reunion mensual de seguimiento. Incluye una seccion de acuerdos, una seccion de riesgos y una seccion de proximos pasos.
```

---

## Opcion G - Convertir una descripcion informal en formato

```text
Necesitamos controlar las tareas que salen de las reuniones, saber quien queda responsable, para cuando debe hacerlo y si esta pendiente o cerrado. Convierte esta necesidad en una plantilla clara.
```

---

## Variante para evaluacion del resultado

```text
Revisa la plantilla que acabas de crear y dime si cumple estos criterios: claridad, facilidad de uso, campos minimos de trazabilidad y necesidad de revision antes de uso oficial.
```

---

## Resultado esperado

El GPT debe devolver una respuesta con:

1. Nombre sugerido del formato.
2. Objetivo.
3. Cuando usarlo.
4. Campos obligatorios.
5. Plantilla editable.
6. Recomendaciones de uso.
7. Nota de revision antes de adoptarlo como formato oficial.
